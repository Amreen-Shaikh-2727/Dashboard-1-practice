# AI Project Landing Page (UI)

A modern and responsive landing page built using HTML and Tailwind CSS. The project features a clean hero section, floating cards, a sticky navigation bar, and a simple SaaS-inspired design.

## Overview

This project includes:

* Sticky navigation bar
* Gradient hero section
* Floating information cards
* Dashboard preview section
* Responsive layout with Tailwind CSS

## Technologies Used

* HTML5
* Tailwind CSS

## Project Structure

```
project/
│── index.html
│── output.css
│── dash.png
│── README.md
```

## How to Run

### 1. Clone the repository

```bash
git clone <repository-url>
```

### 2. Navigate to the project folder

```bash
cd <project-folder>
```

### 3. Install dependencies

```bash
npm install
```

### 4. Build Tailwind CSS


```bash
npx @tailwindcss/cli -i ./src/input.css -o ./output.css --watch
```

### 5. Open the project

Open `index.html` in your browser, or use the **Live Server** extension in Visual Studio Code for automatic reloading.
